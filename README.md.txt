# Network Automation & Data Analysis Projects

Kho lưu trữ này được xây dựng nhằm mục đích thực hành, ứng dụng lập trình Python vào hai mảng công nghệ chính: **Tự động hóa vận hành hệ thống mạng (Network Automation)** và **Phân tích dữ liệu nhật ký hệ thống (Data Analysis & System Logs)**.

---

## 📁 1. Thư mục: `network-automation`
* **Mục tiêu:** Xây dựng các kịch bản lập trình (Scripts) để tự động hóa quản lý cấu hình và giám sát thiết bị mạng từ xa trên môi trường giả lập.
* **Công nghệ & Thư viện sử dụng:** `Python`, `Netmiko`, `Scapy`, `Cisco Packet Tracer`.
* **Trạng thái:** Đang hoàn thiện mã nguồn kịch bản cấu hình tự động.

## 📁 2. Thư mục: `data-analysis`
* **Mục tiêu:** Ứng dụng các thư viện phân tích dữ liệu và thuật toán Học máy không giám sát (Unsupervised Learning) để phân tách cấu trúc tệp dữ liệu log.
* **Công nghệ & Thư viện sử dụng:** `Python`, `Pandas`, `Scikit-learn` (Thuật toán K-Means), `Matplotlib`, `Seaborn`.
* **Trạng thái:** Đang chuẩn bị tập dữ liệu log hệ thống mẫu.
